تفاصيل الكورسات

مقدمة في HTML و CSS

في هذا الكورس تتعلمين أساسيات بناء صفحات الويب باستخدام HTML و CSS. تتعرفين على كيفية إنشاء الهيكل الأساسي للصفحة، إضافة النصوص والصور والروابط، ثم تنسيقها بالألوان والخطوط والمسافات. الكورس مناسب للمبتدئين تماماً، ويعتبر الخطوة الأولى لدخول عالم تطوير الويب.

التصميم المتجاوب

هذا الكورس يركز على جعل موقعك يعمل بشكل ممتاز على الجوال، التابلت، والكمبيوتر. تتعلمين فيه استخدام Media Queries وتقنيات التصميم المتجاوب، وكيفية ترتيب العناصر بحيث تتكيف مع أحجام الشاشات المختلفة بدون أن يتكسر التصميم أو يختفي المحتوى.

تخطيط صفحات الويب باستخدام Flexbox و Grid

في هذا الكورس تتعلمين كيفية ترتيب العناصر داخل الصفحة بشكل احترافي باستخدام Flexbox و CSS Grid. ستعرفين كيف تصممين شبكات للكروت، تقسيم الصفحة لأعمدة وصفوف، وإنشاء تصاميم منظمة وسهلة التعديل، وهي مهارة أساسية لأي مصممة واجهات أمامية.
